## fioctl waves list

Show available waves

```
fioctl waves list [flags]
```

### Options

```
  -h, --help         help for list
  -n, --limit uint   Limit the number of results displayed. (default 20)
  -p, --page int     Page of waves to display when pagination is needed (default 1)
```
